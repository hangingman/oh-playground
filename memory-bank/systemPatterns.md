# システム構成 (System Patterns)

## アーキテクチャ概要
本プロジェクトは OpenHands をベースとした自律開発環境です。

## 主要なコンポーネント
- **OpenHands**: 実行エンジン。自律的なエージェントとして動作します。
- **MicroAgents**: 特定のタスクや規約に基づいた AI の思考をガイドするコンポーネント。
- **Memory Bank**: プロジェクトの長期・短期記憶を保持する Markdown ドキュメント群。

## 重要な設計判断
- **Memory Bank 優先**: Cline などの AI アシスタントは、作業前に必ず Memory Bank を参照し、作業後に更新することで、文脈の喪失を防ぎます。
- **自律的 QA**: 実装と同時にテストコードを作成し、自律的に検証を行うパターンを採用します。
